# Lähtöhälytysäänet

Lisää tähän kansioon MP3-tiedostoja, joita käytetään lähtöhälytyksenä.

## Käyttö:
- Nämä äänet soitetaan kun lähtöaika saavutetaan
- Käyttäjä voi valita haluamansa äänen asetuksista
- Ääni toistetaan N kertaa (määriteltävissä asetuksissa, oletuksena 3 kertaa)

## Suositukset:
- Kiireellinen mutta ei stressaava ääni
- Esim: "Ding ding ding", kellonsoitto, fanfaari
- Pituus: 1-4 sekuntia
- MP3-muoto
- Kohtuullinen äänenvoimakkuus

## Esimerkki:
- departure_bell.mp3
- time_to_go.mp3
- final_alarm.mp3
