# Tehtävän aikahälytysäänet

Lisää tähän kansioon MP3-tiedostoja, joita käytetään tehtävän aikahälytyksenä.

## Käyttö:
- Nämä äänet soitetaan kun tehtävän aika loppuu (aloitusaika + kesto) eikä tehtävää ole merkitty valmiiksi
- Käyttäjä voi valita haluamansa äänen asetuksista
- Ääni toistetaan N kertaa (määriteltävissä asetuksissa, oletuksena 3 kertaa)

## Suositukset:
- Hälyttävä mutta ei pelottava ääni
- Esim: "Beep beep", kellohälytys, summeri
- Pituus: 1-3 sekuntia
- MP3-muoto
- Kohtuullinen äänenvoimakkuus

## Esimerkki:
- alert_1.mp3
- beep_alarm.mp3
- gentle_reminder.mp3
